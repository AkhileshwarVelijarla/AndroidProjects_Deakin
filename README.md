# Android Projects – Deakin University

This repository contains a collection of Android development projects completed as part of my coursework at Deakin University. The projects are built using traditional Android development practices using XML layouts and Java.

---

## 📌 Overview

The purpose of this repository is to demonstrate:
- Understanding of Android fundamentals
- UI design using XML layouts
- Integration of widgets like Spinner, EditText, and Buttons
- Basic application logic and user interaction handling

---

## 🛠️ Technologies Used

- Java  
- Android Studio  
- XML (UI Design)  
- Gradle  

---

## 📂 Project Structure

AndroidProjects_Deakin/
│
├── Project1/
├── Project2/
├── Project3/
└── ...

Each folder represents an individual Android project or task.

---

## 🚀 Features Implemented

- Spinner (Dropdown) integration using ArrayAdapter  
- User input handling with EditText  
- Layout design using LinearLayout / ConstraintLayout  
- Basic validation and event handling  
- Clean and simple UI design  

---

## 📖 Learning Outcomes

Through these projects, I have:
- Built Android apps using structured layouts  
- Learned how to connect UI components with backend logic  
- Understood event-driven programming in Android  
- Practiced writing clean and maintainable code
