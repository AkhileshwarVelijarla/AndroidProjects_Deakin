# Android Projects Deakin

This repository contains three Android coursework projects built with Java and XML in Android Studio. Each project now includes its own `README.md` with architecture notes, data-flow explanations, and the reasoning behind the main implementation choices.

## Projects

### 1. `UnitConversionApplication`

A single-screen unit converter covering currency, fuel efficiency, liquid volume, distance, and temperature conversions.

Project documentation:
- `UnitConversionApplication/README.md`

### 2. `QuizAPP_31C`

A single-activity arithmetic quiz app with a ViewModel-driven state machine, progress tracking, and persisted theme preference.

Project documentation:
- `QuizAPP_31C/README.md`

### 3. `41_PersonalEventPlannerApp`

A small event management app using Navigation, Room, and a shared ViewModel for create, edit, list, and delete workflows.

Project documentation:
- `41_PersonalEventPlannerApp/README.md`

## Shared Technology Choices

- Java for application logic
- XML layouts for UI
- Gradle for builds
- Android Studio as the primary development environment

## Documentation Goal

The repository-level and project-level documentation is written to answer three practical questions:

- how data moves through each app
- why the architecture was kept simple or layered in certain places
- what tradeoffs were made for coursework scope versus long-term scalability

## Running a Project

1. Clone the repository.
2. Open the specific project folder you want in Android Studio.
3. Sync Gradle.
4. Run the app on an emulator or Android device.
